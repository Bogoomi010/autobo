# 편의 기능 구현 결과: 화면 설정 자동 저장

## 구현 내용

- `src/App.tsx`에 사용자 설정 저장 키와 기본값을 추가했다.
- `localStorage`에서 저장값을 읽을 때 타입과 허용값을 검증하도록 했다.
- 다음 값을 앱 시작 시 복원하고 변경 시 자동 저장하도록 했다.
  - 선택 마켓
  - 모의 실행 여부
  - 마켓 필터, 검색어, 정렬 방식
  - 차트 주기
  - 자동 전략 입력값
  - 수동 주문 입력값
- API 키 패널에 `화면 설정 초기화` 버튼을 추가했다.
- `src/App.css`에 초기화 버튼용 보조 스타일을 추가했다.

## 보안 범위

- Access Key와 Secret Key는 저장하지 않는다.
- 잔고, 주문 가능정보, 최근 응답, 로그, 자동 감시 실행 상태는 저장하지 않는다.

## 검증 결과

- `npm run build`: 통과
- `cargo check` in `src-tauri`: 통과

## 비고

- 첫 빌드 시 로컬 `node_modules`가 없어 `tsc`를 찾지 못했다. `npm install` 후 재실행해 통과했다.

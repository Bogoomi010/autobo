# 편의 기능 구현 결과: 수동 주문 빠른 입력

## 구현 내용

- 수동 주문 패널에 `현재가 지정가` 버튼을 추가했다.
  - 선택 종목의 최신 티커 가격이 있을 때 가격 입력칸에 현재가를 채운다.
  - 시장가 매수 금액 칸과 혼동되지 않도록 `ord_type=limit`로 전환한다.
- 자주 쓰는 KRW 시장가 매수 금액 버튼을 추가했다.
  - `10,000`, `50,000`, `100,000`, `500,000 KRW` 버튼을 제공한다.
  - 버튼 클릭 시 현재 선택 종목을 주문 종목으로 맞추고 `side=bid`, `ord_type=price`를 함께 적용한다.
- 수동 주문 요약 문구를 추가했다.
  - 시장가 매수는 매수금액을 표시한다.
  - 지정가 주문은 가격과 수량이 모두 있을 때 예상 주문금액을 표시한다.
- 빠른 입력 영역과 주문 요약 영역의 CSS를 추가했다.
- 샌드박스에서 Vite 기본 config bundling이 상위 디렉터리 접근 제한으로 실패해, `npm run build`가 `vite build --configLoader runner`를 사용하도록 조정했다.

## 검증 결과

- `npm run build`: 통과
- `cargo check` in `src-tauri`: 통과

## 비고

- 이 기능은 기존 사용자 설정 저장 로직의 `manualOrder` 상태를 그대로 사용하므로 별도 저장 키를 추가하지 않았다.
- 계좌 잔고 기반 매도 수량 자동 입력과 호가 단위 보정은 이번 작업 범위에서 제외했다.
